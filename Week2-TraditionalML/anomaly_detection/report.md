# 异常行为识别 — 实验报告

## 1. 问题描述

在仅有正常样本用于训练的情况下，测试阶段识别偏离正常分布的异常样本。使用合成数据集（3 个高斯簇 + 5% 随机异常），评估 Isolation Forest 和 One-Class SVM 的检测能力。

## 2. 训练策略

两种模型均采用 **仅用正常样本训练** 的策略（无监督异常检测的标准范式）：
- 训练集：仅正常样本
- 测试集：正常 + 异常混合（含 30% withheld 数据）

## 3. 模型对比

| 指标 | Isolation Forest | One-Class SVM |
|---|---|---|
| Accuracy | **95.56%** | 92.06% |
| AUC | **0.9600** | 0.8633 |
| Precision | 1.0000 | 0.3529 |
| Recall | 1.0000 | 0.8000 |

## 4. 分析

- **Isolation Forest 全面优于 One-Class SVM**：采用随机划分树的集成策略，在合成数据的高斯簇异常检测中表现完美（Precision=Recall=1.0）
- **One-Class SVM 的 Precision 较低 (35.29%)**：RBF 核的超球面边界过于宽松，将部分簇间空白区域误判为正常
- **决策边界差异显著**：IF 的边界紧贴数据分布轮廓，OC-SVM 的边界更平滑但覆盖率更广
- **IF 的计算效率也更高**：无需计算成对距离矩阵，在更大的数据集上更有优势

## 5. 结论

Isolation Forest 在合成数据的异常检测中表现优异（95.56% 准确率），且计算开销小。One-Class SVM 虽然经典，但在高维和簇状分布数据上容易出现 Precision 偏低的问题。
