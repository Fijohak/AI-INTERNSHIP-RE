# California Housing 线性回归 — 实验报告

## 1. 问题描述

基于 California Housing 数据集的 8 个特征（收入、房龄、房间数、地理位置等），预测街区房价中位数。回归任务，20,640 条数据。

## 2. 数据预处理

- **标准化**：`StandardScaler` 将 8 个特征标准化到均值 0、标准差 1
- **划分**：80/20 训练/测试划分 (random_state=42)
- **目标分布**：原始房价呈右偏分布，采用 `log1p` 变换后接近正态

## 3. 模型

**Linear Regression** — 最基本的线性回归模型，假设目标值与特征之间存在线性关系。

由于任务是纯线性回归（非对比实验），仅使用一个模型，重点验证完整的回归分析流程。

## 4. 实验结果

| 指标 | 值 |
|---|---|
| R² | **0.5758** |
| MSE | 0.5559 |
| RMSE | 0.7456 ($74,560) |
| MAE | 0.5332 ($53,320) |

## 5. 分析

### 5.1 残差分析
- 残差近似正态分布，均值为 0，满足线性回归的基本假设
- 预测值较高时残差有轻微发散趋势，可能存在轻微的异方差性

### 5.2 特征权重
- **MedInc**（收入中位数）是最重要的正向特征，系数远大于其他特征
- **Latitude** 和 **Longitude** 具有显著的负系数（反映加州房价的地理梯度）
- **AveOccup** 负向影响房价：人均居住面积越小，房价越高

### 5.3 模型局限性
- R²=0.5758 说明线性模型仅能解释约 58% 的方差
- 房价受非线性因素（地段、政策、社区设施）影响，纯线性模型能力有限
- 改进方向：决策树回归、随机森林回归、XGBoost 回归

## 6. 结论

线性回归在 California Housing 上取得了 R²=0.5758 的结果，符合简单线性模型的预期。特征权重分析表明收入中位数（MedInc）是房价最重要的预测因子，与常识一致。
